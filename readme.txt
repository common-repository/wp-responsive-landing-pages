=== WP Responsive Landing Pages  ===
Tags: WP Responsive Landing Pages, Responsive Landing Page, Landing Page, WP Responsive Page
Requires at least: 4.3
Tested up to: 4.8

== Description ==

This plugin allows an easy institution of responsive landing pages to be run on your WordPress website. An easy to manage interface, multiple options to modify every element and real-time updates, it makes the process simple and easy.

= Salient Features Of WP Responsive Landing Pages: =
- Add separate / full template
- Drag & Drop modules
- Support for SEO elements
- Analytics settings
- Easy follow up form with integrated email settings
- Fully responsive landing page
- Easily modify look & feel using multiple design options

This plugin has been developed to make the landing page creation easy for the store owners. Any suggestions or feature requests are welcome! 

= Usage Guide =
- Click on 'add new' sub menu of Landing pages link
- You can choose full template or you can also choose module according to our need.
- Make required customization in the text & styling settings & publish it
- With the help of preview button you see the preview of the page.

== Installation ==
- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select 'Add New' from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress
- Add image collages on the page or post screen

Or 

- Upload the WP Responsive Landing Pages directory (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress
- Add template to begin creating a responsive landing page for your website
